# 🌐 Laravel Portfolio Web Application

A modern and professional portfolio web application built using **Laravel 12**, designed to showcase projects, education, experience, and resume with a clean admin panel.

---

## 🚀 Features

### 🔹 Frontend (Public Website)
- Home page with premium UI (beige theme)
- About section
- Dynamic Projects listing
- Dynamic Education & Experience
- Resume download option
- Contact form
- Responsive design using Bootstrap

---

### 🔹 Admin Panel
- Secure admin authentication (Laravel Breeze)
- Manage Projects (CRUD with image upload)
- Manage Education details
- Manage Experience details
- Upload and update Resume
- Dashboard with statistics
- Clean and modern UI

---

## 🛠️ Tech Stack

| Technology | Usage |
